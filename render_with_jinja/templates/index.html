Playing with PYNET's Jinja Implementation

This website can be used to practice building and designing good templates.

You can load in templates to extend from into the database, then you may write your own templates that could be the test case body request/response. When rendering, input a jinja map and the type of final content you are rendering (JSON, XML, Neither). The template will be rendered, and validated if the type selected is XML or JSON.

See the PYNET documentation on templating and the Jinja Designer Template Documentation for tips and tricks designing templates.
